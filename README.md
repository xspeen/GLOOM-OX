🎮 COMMANDS & USAGE

QUICK START

Command Action
python gloom-ox.py Launch interactive mode
python gloom-ox.py "URL" Direct download
python gloom-ox.py --update Update all dependencies
python gloom-ox.py --dir Show download folder
python gloom-ox.py --version Show version

INTERACTIVE COMMANDS

Command Function
clear Clear terminal screen
dir List downloaded files
history Show download history
update Force update dependencies
robot Launch assistant
exit / q Exit GLOOM-OX
