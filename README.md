## INSTALLATION

**One-Line Installer (Linux/macOS/Termux)**
```bash
curl -sSL https://raw.githubusercontent.com/xspeen/GLOOM-OX/main/scripts/install.sh | bash
```

One-Line Installer (Windows PowerShell as Admin)

```powershell
iex (irm https://raw.githubusercontent.com/xspeen/GLOOM-OX/main/scripts/install.ps1)
```

Docker

```bash
docker run -it --rm xspeen/gloom-ox:latest
```

Manual Install

```bash
git clone https://github.com/xspeen/GLOOM-OX.git
cd GLOOM-OX
pip install -r requirements.txt
python gloom-ox.py
```

<br>

QUICK START

```bash
# Interactive Mode
python gloom-ox.py

# Direct Download
python gloom-ox.py "https://youtube.com/watch?v=VIDEO_ID"

# Batch Download from File
python gloom-ox.py --batch urls.txt
```

COMMANDS

Command Action
python gloom-ox.py Launch interactive mode
history Show downloaded videos
clear Clear terminal screen
dir List downloaded files
exit or q Exit program
