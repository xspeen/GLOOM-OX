🔹 Step‑by‑Step for Each OS

<details>
<summary><b>🐧 Linux (Ubuntu/Debian/Arch/Parrot)</b></summary>

```bash
# 1. Install dependencies
sudo apt update && sudo apt install python3 python3-pip ffmpeg git -y   # Debian/Ubuntu
# or for Arch/Parrot:
sudo pacman -S python python-pip ffmpeg git

# 2. Clone GLOOM-OX
git clone https://github.com/xspeen/GLOOM-OX.git
cd GLOOM-OX

# 3. Install Python packages
pip3 install -r requirements.txt

# 4. Run
python3 gloom-ox.py
```

</details>
